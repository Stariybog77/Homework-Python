# Лабораторная работа №3 (Рекурсия, Python)

## Условия задач

Задание 1  
Реализовать функцию для вычисления суммы элементов вложенного списка.  
Необходимо реализовать два варианта:
- с использованием рекурсии
- без использования рекурсии

Пример:
sum_nested([1, [2, [3, 4, [5]]]]) = 15

---

Задание 2  
Реализовать функции для вычисления последовательностей aₖ и bₖ:

aₖ = 1/2 (√bₖ₋₁ + √aₖ₋₁)  
bₖ = 3/2 √bₖ₋₁ + 1/2 aₖ₋₁²  

Начальные значения:
a₁ = 1  
b₁ = 1  

Требуется реализовать:
- решение с рекурсией
- решение без рекурсии

---
