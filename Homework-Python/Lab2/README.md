# Лабораторная работа №2 (Python)

## Условия задач

Задание 1  
Ольга составляет таблицу кодовых слов. Используются 4-буквенные слова из букв A, B, C, D, X, Y, Z.  
Первая буква — X, Y или Z, остальные — только A, B, C, D.  
Определить количество таких слов.

Задание 2  
Вычислить значение выражения:
9^8 + 3^5 - 9  
Представить результат в системе счисления с основанием 3 и найти количество цифр 2.

Задание 3  
Найти все натуральные числа на отрезке [40000; 50000], у которых ровно 5 различных нечётных делителей.
