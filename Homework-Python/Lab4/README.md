# Лабораторная работа №4 (Замыкания и декораторы, Python)

## Условия задач

Задание 1  
Реализовать замыкание для получения текста ответа на запрос к API.  
В качестве примера используется API: https://dogapi.dog/api/v2/facts  

Задание 2  
Реализовать декоратор, ограничивающий частоту вызова функции.

---

## Описание проделанной работы

В ходе выполнения лабораторной работы:

- реализовано замыкание, возвращающее функцию для выполнения API-запроса
- использован внешний API для получения случайных фактов о собаках
- реализован декоратор для ограничения частоты вызова функции
- декоратор применён к функции, выполняющей API-запрос

### Использованные технологии:

- функции Python
- замыкания (closure)
- декораторы
- работа с HTTP-запросами (библиотека requests)
- модуль time для ограничения частоты вызова
