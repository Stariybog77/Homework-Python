## Задание

Выполнить 11 заданий по Python:

1. Расчёт расстояний между городами  
2. Площадь круга и проверка точки  
3. Арифметическое выражение  
4. Работа со строками (срезы)  
5. Работа со списками  
6. Работа со списками (зоопарк)  
7. Работа со списками и словарями  
8. Расшифровка строки  
9. Работа с множествами  
10. Вложенные словари  
11. Подсчёт стоимости товаров

## Шпаргалка по Git

Инициализация:
git init

Добавление файлов:
git add .

Коммит:
git commit -m "Добавлены задания lab1"

Подключение репозитория:
git remote add origin https://github.com/USERNAME/Homework-Python.git

Отправка:
git push -u origin main

Проверка:
git status
